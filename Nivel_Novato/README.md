# ♟️ Desafio: Movimentando as Peças de Xadrez em C

Este projeto consiste em um simulador de movimentação de peças de xadrez (Torre, Bispo e Rainha) desenvolvido em linguagem C. O objetivo principal é demonstrar a aplicação prática de diferentes estruturas de repetição para percorrer casas em um tabuleiro hipotético.

## 🚀 Objetivo

Aplicar os conceitos de lógica de programação utilizando as estruturas de repetição `for`, `while` e `do-while`, garantindo que cada peça siga sua regra específica de movimento conforme as instruções do desafio.

## 🛠️ Funcionalidades
O programa simula os seguintes movimentos:

**Torre**: Move-se 5 casas para a **Direita** utilizando a estrutura `for`.

**Bispo**: Move-se 5 casas na diagonal (**Cima e Direita**) utilizando a estrutura `while`.

**Rainha**: Move-se 8 casas para a **Esquerda** utilizando a estrutura `do-while`.

## 📋 Requisitos do Sistema

*   Compilador C (GCC, Clang, MSVC, etc.)
*   Ambiente de execução de terminal (Linux, macOS ou Windows Prompt/PowerShell)
